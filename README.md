# Chip8OdinEmu

A Chip-8 emulator written in Odin

> Requires [minifb](https://github.com/emoon/minifb) (*MIT License*) - A pixel framebuffer library - to be compiled into `minifb.lib` (Windows) or `minifb.a` (Posix-based) and stored at `lib/`.
>
> Bindings are manually created under `src.*/minifb/` translating the headers the repository contains
